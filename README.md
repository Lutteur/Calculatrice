
# Calculatrice Helldivers

Projet : Une calculatrice codée en HTML, CSS et JavaScript qui effectu des opérations classiques.

## Pourquoi Helldivers ?

J'ai nommée mon projet "Calculatrice Helldivers", tout simplement parce que mon jeu préférée est Helldivers 2, qui est un jeu coopératif et de tir a la 3ème personne. J'ai donc décider de faire le CSS de mon projet avec les couleurs du jeu vidéo.

## Les Versions ?
Comme vous pouvez le constatez j'ai crée deux versions, une version personnel et une version universel. C'est tous a fait normal, les deux version marchent de la même façon, je vais vous expliquez pourquoi.

## Version Personnel :

J'ai en effet crée une version Personnel, à cause du PC sur lequel j'ai développer cette calculatrice. Ce Pc portable n'a pas d'emplacement spécifiques pour entrer uniquement des numéros, ce que je veux dire c'est que les numéros sont sur les mêmes touches que des caractères spéciaux comme le "é", "&" etc. Sachant que j'utilise dans le code JavaScript j'utilise des KeyPads pour détecter sur quel touche l'utilisateur appuie, il est donc logique que je n'ai pas pu faire fonctionner pareil sur deux versions de claviers différents.
