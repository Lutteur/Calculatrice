# Calculatrice

Projet : Une calculatrice codée en HTML, CSS et JavaScript qui effectu des opérations classiques.

Comme vous pouvez le constatez j'ai crée deux versions, une version personnel et une version universel.
C'est tous a fait normal, les deux pages marchent de la même façon, je vais vous expliquez pourquoi.

Version Personnel :

J'ai en effet crée une version Personnel, a cause du PC sur lequel j'ai développer cette calculatrice.
Ce Pc portable n'a pas d'emplacement spécifiques pour entrer uniquement des numéros, ce que je veux dire
c'est que les numéros sont sur les mêmes touches que des caractères spéciaux comme le "é", "&" etc.
